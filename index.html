Projects

    Nestle DVIR App

    Mobile App Challenge 2019. Together in a group of four, our goal was to create an app that simplified the truck inspection process for Nestle truck drivers. We converted the Driver Vehicle Inspection Report (DVIR) from a physical pen-and-paper form into a digital app accessible by smart phones. The app includes additional features that made use of the camera to not only allow users to take pictures of their vehicle, but also ensure that they fill out their forms correctly.

    This app was created using Angular/Ionic and coded in TypeScript.

    SQL Database Project. This project was created in a team of two. The purpose of this project was to demonstrate user ineractions with a database. A simple interface was created to allow the user to easily read and modify data.
